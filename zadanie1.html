a) Uwagi na Temat JS:

1. Javascript powstał 4 grudnia 1995

2. JavaScript stanowi podstawowe narzędzie do tworzenia stron internetowych

3. Javascript po stronie klienta jest wykonywany w kompilatorze javascript

4. JavaScript ma pewne ograniczenia, takie jak uniemożliwienie odczytu i zapisu na dysku oraz uniemożliwianie dostępu do innych okien lub domen przeglądarki

5. JavaScript jest językiem obiektowym , ponieważ umożliwia lub wymusza stosowanie obiektowych metod programowania.
